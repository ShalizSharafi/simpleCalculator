## Features

- Addition, subtraction, multiplication, and division
- Live operator symbol display between inputs
- Input validation with visual feedback
- Division by zero protection
- Sound feedback on operator selection
- Keyboard support — press `Enter` to calculate
- Clean dark UI with a chartreuse accent

---

## Usage

No setup needed. Just open `index.html` in your browser.

```bash
git clone https://github.com/your-username/calculator.git
cd calculator
open index.html
```

---

## How It Works

Pick an operator → fill in both numbers → hit **Calculate** (or press `Enter`).

| Input | Action |
|---|---|
| Click `+` `-` `÷` `×` | Sets the operator |
| Type in both fields | Sets the numbers |
| Click Calculate or press Enter | Outputs the result |
